Interpolators

Principle

Interpolators are VRML nodes that provide an output value depending on

  • an input value from 0 to 1
  • a list of keys

A ROUTE from TimeSensor TO input value of interpolator starts the calculation of the output value. A ROUTE from the output value of interpolator TO a field of another node will animate this field.


There is one kind of interpolator per field type. There are several supported interpolators in the MeMo:

  • ColorInterpolator
  • CoordinateInterpolator
  • CoordinateInterpolator2D
  • OrientationInterpolator
  • PositionInterpolator
  • PositionInterpolator2D
  • ScalarInterpolator

Common interface

All these interpolators share a common interface and behavior.


Every interpolator has the following fields:

  • the set_fraction field
  • the value_changed field
  • the key field
  • the keyValue field
