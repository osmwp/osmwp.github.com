Shape node


Warning:
Geometry property does NOT support the DEF / USE.


Draw a shape with the specified appearance and geometry.


SFNode appearance NULL :
Mandatory [Appearance] node.

SFNode geometry NULL :
A geometry node.

List of geometry nodes available in a 2D context:

  • Bitmap
  • Circle
  • IndexedFaceSet2D
  • Rectangle
  • RichText
  • Text
  • WrapText
