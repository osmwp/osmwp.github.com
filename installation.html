Installation

To ease the Mobile Widgets development, the SDK is integrated with Eclipse as a plug-in.

The SDK contains:

  • Mobile Widgets Player (MeMo Player) to execute widgets.
  • GUI generic components: supplies user interface elements (list, menu bar, etc.).
  • Mobile phone emulator: a virtual mobile device to design, debug and test widgets.
  • Samples: basics demos to start easily development.

Mobile Widgets development environment is based on Java and Eclipse.

This part aims to describe the installation and the configuration of Java runtime, Eclipse and Mobile Widgets SDK.
