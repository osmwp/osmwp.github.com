VRML nodes reference

Introduction

Below is the list of all supported nodes with the implemented fields and their applicable context.
Do not try to use a node in a context it does not support.

The nodes have been implemented to follow as much as possible the VRML / MPEG-4 specifications.
However any deviation, addition or difference with the standards are mentioned or described.
