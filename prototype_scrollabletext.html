ScrollableText prototype

Scrollable text is composed of 3 parts: a title, an image and the text.
It is based on the Scrollable prototype to support scrolling.



Data fields

  • MFString title - Title, can have multiple lines
  • MFString images - Url of the image associated to the text content (to be scrolled)
  • MFString content - Text content (to be scrolled)

Style fields

  • MFString style - Path to the CSS node associated to ScrollableText proto
  • MFString contentStyle - Path to the CSS specific node associated to the content of ScrollableText proto
  • MFString titleStyle - Path to the CSS specific node associated to the title of ScrollableText proto
  • SFVec2f size - Size of the area where the ScrollableText proto is displayed
  • SFVec2f textScale - Use for scaling text if necessary (phone with small fonts)
  • SFInt32 titleMaxLines - Title max lines
  • SFInt32 contentMaxLines - Content max lines
  • SFBool enable - If FALSE, key and touch events will be ignored. Default: TRUE.

Warning:
Define style is required (we can use the default style - see Style node).
