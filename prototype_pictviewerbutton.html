PictViewerButton prototype

Introduction

Images which vary with the state.


Interface


Fields description

Event in

  • SFTime pressed - Event when image is pressed. Update the button state.
  • SFTime released - Event when image is released. Update the button state.

Data fields

  • SFBool enable - Is image enabled ?
  • MFString images - List of images. Images order: none, normal, pressed, locked
  • SFVec2f pos - Position
