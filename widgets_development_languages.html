Mobile Widgets languages

Mobile Widgets is based on MeMo Player to displayed widgets.

To describe scenes, MeMo use 3 languages:

  • VRML language
  • Javascript
  • CSS

VRML

VRML (Virtual Reality Modeling Language) is a standard file format for representing 3D scenes. MeMo player extends VRML for representing 2D scenes.

More information about VRML: see http://en.wikipedia.org/wiki/Vrml

To start, read the VRML Totally Beginner's Guide.

List of VRML nodes: VRML nodes reference.

List of Mobile Widgets prototypes: prototypes chapter.


Javascript

Javascript is used by VRML to add more interactivity in VRML scene.

More information about Javascript language: http://en.wikipedia.org/wiki/Javascript
