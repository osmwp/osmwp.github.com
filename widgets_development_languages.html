Mobile Widgets languages

Mobile Widgets is based on MeMo Player to displayed widgets.

To describe scenes, MeMo use 3 languages:

  • VRML language.
  • JavaScript.
  • CSS.

VRML (Virtual Reality Modeling Language) is a standard file format for representing 3D scenes.
MeMo (Media Mobile) player extends VRML for representing 2D scenes.

To have more information about:

  • VRML, see http://en.wikipedia.org/wiki/Vrml.
  • starting VRML, read the VRML totally beginner's guide chapter.
  • the list of VRML nodes, see VRML nodes reference chapter.
  • the list of Mobile Widgets prototypes, see prototypes chapter.

JavaScript is used by VRML to add more interactivity in VRML scene.

To have more information about:

  • JavaScript language, see http://en.wikipedia.org/wiki/Javascript.
  • JavaScript in MeMo, see JavaScript introduction chapter.
  • list of JavaScript libraries, see JavaScript reference chapter.

MeMo use CSS (Cascading Style Sheets) language to change the application style.

To have more information about:

  • CSS language, see http://en.wikipedia.org/wiki/CSS.
  • CSS in MeMo, see CSS language chapter.
