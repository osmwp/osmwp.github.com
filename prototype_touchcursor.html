TouchCursor prototype

This prototype simulates an hardware cursor using a touchscreen.



EventOut

  • SFTime eventTime - Time when the user releases the touchscreen.
    This event is only triggered if a valid activeKey could be computed.
  • SFString activeKey - The virtual key that has been detected (one of "RULED").
  • SFBool dragged - True on a dragging event, false on a clicked event.
  • SFVec3f hitPoint_changed - Event of touch routed from TouchSensor

Fields

  • SFString url - List of keys events to handle. Default: "RULED".
  • SFBool activate - If false, the prototype will not send events. Defaut: TRUE.
  • SFVec2f size - Size of the touchscreen zone used by the prototype. Defaut: 0 0.
  • SFString mode - Same as mode field of TouchSensor.
