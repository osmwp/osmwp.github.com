deployment on phone

Introduction

Deploying the application to a phone can vary between phone.

Note:
This section addresses only the deployment via USB cable.

Requirements

Install widget on mobile require:

  • A mobile with Java support
  • An USB cable to transfert widget on mobile
  • The widget JAR and JAD files

How to run a widget on mobile phone

Connect your computer and your mobile via the USB cable.
Your mobile should be recognized by the computer.
Otherwise, please refer to the documentation for your phone.

Copy JAR and JAD files on your mobile.

On mobile, open file manager (depends of mobile phone). Go to the folder where JAR and JAD files are copied.
Select en launch the JAR file.

Warning:
Don't launch JAR file.
