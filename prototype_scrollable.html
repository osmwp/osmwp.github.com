Scrollable prototype

Introduction

Scrollable is a proto used to scroll generic content.
It displays a scroll bar if the content is too long for its size.


Interface


Fields description

Fields

  • MFNode children - Content to be scrolled.be Mandatory field.
  • SFVec2f size - Size of the scrollable proto. Mandatory field.
  • SFVec2f contentSize - Size of the content to scroll. Mandatory field.
  • SFBool enabled - If FALSE, key and touch events will be ignored. Default: TRUE.

Style

  • MFString style - Path to the CSS node associated to Scrollable proto

Warning:
Define style is required (we can use the default style - see Style node).
