prototypes

Mobile Widgets offers many reusable prototypes.
With prototypes, we can have an user interface similar between applications.

There are multiple prototypes:

  • Alert: An alert popup message
  • Bubble: Display a message in a bubble
  • Loader: Gray the screen and display a message and a loading animation
  • ...

Tiny Mobile Widgets offers less reusable prototypes than Mobile Widgets. The reason is that some of the prototypes above need a high CPU consumption. In Tiny Mobile Widget, HMI should also be simplier than in Mobile Widget because of the small screen size of targeted devices.

Tiny Mobile Widget also uses a scene that is not a prototype (in the VRML sens). It is called adapter/manager and provides all the menu, title and softkeys management.
